# EZQR

**[ezqr.ca](https://ezqr.ca)** - a free QR code generator. Make a static QR code in seconds (no account, watermark, or expiry), or [sign up](https://ezqr.ca) for dynamic QR codes you can edit after printing and track scans on.

## Open source

We publish a few MIT-licensed projects so you can drop the EZQR generator into your own site or self-host it.
